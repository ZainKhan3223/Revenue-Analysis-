# Project Revenue Engine

An AI-powered revenue optimization dashboard for small businesses.

## Features
- **Predictive Analysis**: High-density forecasting rows with dynamic growth modeling.
- **Revenue Mix**: Interactive donut charts for product lines.
- **Executive Overview**: Premium Area Charts with multi-stop gradients and glow effects.
- **Liquidity Monitoring**: Cash flow bars with real-time net monitoring and visual glow.
- **AI Recommendations**: Confidence-based strategies for revenue optimization.

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS, Recharts, Lucide/Material Icons.
- **Backend**: FastAPI, Scikit-learn, Pandas.
- **DevOps**: Docker & Docker-Compose.

## Getting Started
1. **Repository Setup**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
2. **Launch Application**:
   ```bash
   docker-compose up --build
   ```
3. **Access Dashboard**: Open `http://localhost:3000`
